# Data Notice

This public repository does not include raw medical imaging data.

## Expected local data layout (example)
- `data/raw/dicom/...`
- `data/raw/nifti/...`
- `data/roi/...`

## Requirements
- Use only de-identified and authorized data.
- Do not commit raw DICOM, private metadata, or patient identifiers.
- Keep large intermediate outputs outside this repository.
