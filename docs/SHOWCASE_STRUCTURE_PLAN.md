# SHOWCASE_STRUCTURE_PLAN

## Minimal public structure
- `README.md`
- `.gitignore`
- `docs/`
- `nsclc_swinunetr/scripts/` (core only)
- `nsclc_swinunetr/configs/` (core only)
- `datalist/` (lightweight only)
- `scripts/run_demo_placeholder.md`
- `demo/` (curated assets only, optional to add now)

## Keep/avoid principle
- Keep: method flow, configuration templates, small demo artifacts.
- Avoid: raw data, bulk outputs, duplicated legacy snapshots.
